## Распределение адресного пространства
##### Адресное пространство будет распределяться по формуле:
ip = 10.st.splf.x, где:
1. st - номер ЦОДа
2. sp - номер Spine/Leaf
3. x - по порядку
4. 0 - reserved
5. 1 - loopback1
6. 2 - loopback2
7. 3 - reserved
8. 4 - 7 - ptp1
9. 8 - 11 - ptp2
10. 12 - 15 - reserved
11. 16 - 31 - hosts network
